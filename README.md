# emote-generator

Build emotes with ease.

## UI

### Build

Outputs to `./generator/dist`

```bash
npm run build
```

### Develop

Serves with Vite on http://localhost:3000

```bash
npm run dev
```

## API

### Build

Outputs to `./dist`
Serves with Express on http://localhost:4000

```bash
npm run build
npm run start
```

### Develop

Serves with Express on http://localhost:4000

```bash
npm run dev
```

### Lint

```bash
npm run lint
```
